# Screenshot Guide

## 中文

建议你后续把 GitHub 仓库首页截图放到：

- `docs/images/main-home.png`
- `docs/images/template-editor.png`
- `docs/images/canvas-designer.png`
- `docs/images/print-preview.png`

1. 主界面
2. 模板编辑器
3. 画布设计器
4. 打印预览

如果后续你补了截图，可以在 `README.md` 里加：
