# GitHub Release Template

## 中文

### 标题示例

`v1.0.0 Public Build`

### 发布说明示例

```text
公开版更新内容：

1. 提供独立公开版源码
2. 去除管理员端、授权端、云管理、服务器中转
3. 支持用户自行填写 Sign API Base 和 Sign API Key
4. 保留本地监听、模板编辑、画布设计和打印功能

适用说明：
- 本版本为本地公开版
- 不依赖后台授权系统
- 不影响正式商用版本
```

## English

### Example Title

`v1.0.0 Public Build`

### Example Release Notes

```text
Public build updates:

1. Independent public source release
2. Removed admin, licensing, cloud management, and server relay
3. Users can enter their own Sign API Base and Sign API Key
4. Keeps local listening, template editing, canvas design, and printing

Notes:
- This is a local public build
- No backend licensing system is required
- It does not affect the production/commercial build
```
